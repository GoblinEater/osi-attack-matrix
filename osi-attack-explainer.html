MITRE CWE — Common Weakness Enumeration
Community-developed list of software and hardware weakness types that serve as a common language for security tools and services.
cwe.mitre.org

MITRE CAPEC — Common Attack Pattern Enumeration
Comprehensive dictionary of known attack patterns used by adversaries to exploit weaknesses in cyber-enabled capabilities.
capec.mitre.org

CISA Known Exploited Vulnerabilities (KEV)
Authoritative catalog of vulnerabilities that are being actively exploited in the wild, maintained by CISA.
cisa.gov/known-exploited-vulnerabilities-catalog

NIST National Vulnerability Database
U.S. government repository of standards-based vulnerability management data using SCAP.
nvd.nist.gov

Standard awareness document for web application security representing the most critical security risks.
owasp.org/www-project-top-ten

NIST SP 800-53 Rev. 5 — Security Controls
Catalog of security and privacy controls for information systems and organizations.
csrc.nist.gov

Knowledge graph of cybersecurity countermeasures, mapping defensive techniques to ATT&CK offensive techniques.
d3fend.mitre.org
